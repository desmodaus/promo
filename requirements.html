Стек та практики

Технології, які використовую щодня

Я працюю з JavaScript, React, React Hooks, TypeScript та Redux Toolkit, а також додаю React Router, MUI, REST API, Git і інструменти якості коду. Принципи SOLID, DRY, KISS, YAGNI — обов'язкова основа.

Мій стек

  • JavaScript для базової логіки та інтеграцій.
  • React + Hooks для компонентів і стану.
  • TypeScript для стабільних контрактів.
  • Redux Toolkit для керування глобальним станом.
  • React Router для структури сторінок.
  • MUI та CSS для інтерфейсів.

Принципи, які підтримую

Вони допомагають зберігати простоту, масштабованість і чистоту коду.

SOLID

Розділення відповідальностей і прозора архітектура.

DRY

Жодного дублювання: повторюване — виношу в модулі.

KISS

Складне роблю простим, читабельним і підтримуваним.

YAGNI

Додаю функції лише тоді, коли вони реально потрібні.

ООП

Структурую логіку так, щоб її легко було розширювати.

Тести

Перевіряю критичну логіку та ключові сценарії.
